# Mobile Style Refinement Execution

Этот execution-doc фиксирует новый активный tranche по визуальной доводке `fit` после общей функциональной стабилизации.

Цель:
- убрать ощущение тяжёлых и растянутых блоков на телефоне;
- привести вход, shell и ключевые user surfaces к более компактной и коммерчески понятной mobile-подаче;
- отдельно уплотнить `/nutrition`, потому что там всё ещё слишком много вертикального воздуха и крупных поверхностей.
